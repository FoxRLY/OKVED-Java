# Игра "Найди ОКВЭД по номеру телефона"
Тестовое задание
## Необходимые зависимости:
- Java 21
## Как запускать?
```./gradlew run --args="+79999999999"```
## Пример работы:
```
./gradlew run --args="+79999999901"

Телефон: +79999999901
Код ОКВЭД: 01
Название: Растениеводство и животноводство, охота и предоставление соответствующих услуг в этих областях
Совпадение: 2
```
